# La Petite Garoupe Website

Static Vite/React website scaffolded for the La Petite Garoupe reopening preview.

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run validate:locales
npm run build
```

## GitHub Pages

The temporary preview target is:

```text
https://weelsapp.github.io/la-petite-garoupe/
```

Build the static preview with:

```bash
npm run build:pages
```

When the real domain is ready, update `VITE_SITE_ORIGIN`, the GitHub Pages custom domain settings, and the final contact/legal details.
